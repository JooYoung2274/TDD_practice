## TDD_practice.

- 간단한 CRUD TDD 연습

### 1. 기술 스택

- Typescript
- Express
- Mongoose
- typedi
- jest
- supertest

### 2. 로컬 실행 방법

```
git clone https://github.com/JooYoung2274/TDD_practice.git
cd TDD_practice
npm install

--> .env 파일 내의 DB_URL 변경. (본인 로컬 mongoDb url)

npm test
```
